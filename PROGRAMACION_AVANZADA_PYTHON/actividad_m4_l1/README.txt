• ¿Qué te pareció más interesante o diferente respecto a cómo habías programado antes?

    Lo que me pareció más interesante fue que ahora el código se organiza en objetos que representan cosas reales, en vez de solo usar funciones sueltas. También fue diferente pensar primero en las clases y sus responsabilidades,lo que hace que el programa sea más ordenado y fácil de entender.
