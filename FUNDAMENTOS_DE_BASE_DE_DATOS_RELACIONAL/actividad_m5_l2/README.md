
• ¿Qué es un modelo de datos y para qué sirve en bases relacionales?

    Un modelo de datos es la forma en que se organiza la información dentro de una base de datos. Define qué tablas existen qué campos tiene cada una y cómo se relacionan entre sí, 1 a 1, 1 a muchos y muchos a 1.

• ¿Qué es una clave foránea y qué garantiza?

    Una clave foránea es un campo de una tabla que se conecta con la clave primaria de otra tabla para relacionar los datos.

    Garantiza que:

        -No se puedan ingresar datos que no existan en la tabla relacionada.
        -No existan registros sin relación
        -Se mantenga la integridad de la información
