¿Cuál fue la diferencia principal que notaste entre for y while?

    La principal diferencia diria que en el for esta restringido para usar en rangos o listas, en cambio el while lo puedes ocupar mas libremente según el ciclo 
    que quieras realizar.

¿Te pareció útil el uso de break y continue? ¿Por qué?

    Me parecio útil, imagidandome los usos que se le darian en un entorno real podria ser caso de error el continue permita seguir ejecutando el problema,
    para no provocar interrupcion de un servicio y que este luego pueda ser gestionado a traves de un log que nos permite conocer a detalle el error que sucedio.
    El break quizas para una condicion mas borde, errores criticos o para temas de ciberseguridad proteger datos podria servir en algun algoritmo para que no haya fugas de 
    activos valiosos para una empresa.
