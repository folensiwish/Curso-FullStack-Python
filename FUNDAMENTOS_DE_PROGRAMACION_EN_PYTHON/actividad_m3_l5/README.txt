¿Cuál estructura te pareció más flexible?

    La lista, es una estructura que es mas flexible, tiene metodos incorporados para llamar como append, remove,sort. Tambien te permite imprimir elementos, 
    extraerlos individualmente para hacer operaciones, para mezclarlos, ordenarlos, cambiar de posicion etc..

¿Para qué tipo de problema usarías un diccionario en lugar de una lista?

    Le daria un diccionario a valores que tengan una llave para su identificacion o tambien para crear objetos y poder utilizar sus distintos atributos 
    asi podemos tener varios objetos con distintos valores. Como por ejemplos las listas se clasificiaria para solo un atributo donde los elementos deben ser 
    de la misma categoria.
